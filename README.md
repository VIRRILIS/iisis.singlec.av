Сайт проекта: http://av.iisis.ru

модуль av.singlec разработан для платформы 1С 8.х и используется для более удобного и полного поиска запасных частей и их аналогов для автомобилей.

av.singlec реализован с использованием фреймворка iis.singlec, что позволяет модулю адаптироваться к большинству конфигураций, написанных на 1С 8.x

Источниками получения информации о деталях могут служить:

Любые сайты, где есть информация о продукции;
Веб-сервисы, предоставляемые поставщиками или производителями деталей;
Текстовые файлы прайс листов (Excel, txt, csv и другие);
Сторонние базы данных (Н-р: TecDoc);
Любой другой источник данных;
